# Scope

- node: docs/overview/principles/scope.md
  - in_scope:
    - runtime JSON config loading
    - command surface tied to the achievement catalog
    - GUI menus for the achievement catalog
    - villager dialogue runtime and quest acceptance flow
    - player affinity and quest state persistence
    - canonical authority is the source tree plus this spec tree
  - out_of_scope:
    - skill systems
    - compatibility migrations for old save formats
  - non_authoritative_artifacts:
    - build/ outputs
    - generated sources and resources
